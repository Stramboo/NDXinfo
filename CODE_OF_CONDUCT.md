# 贡献者行为准则

## 我们的承诺

为了营造一个开放、友好、包容的社区环境，我们作为贡献者和维护者承诺：无论年龄、体型、残障、种族、性别认同与表达、经验水平、国籍、个人外貌、宗教、性取向或社会经济地位如何，参与本项目和社区都将免受骚扰。

## 我们的准则

有助于创造积极环境的行为包括：

- 使用友好和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表达同理心

不可接受的行为包括：

- 使用性暗示语言或图像，以及不受欢迎的性关注或挑逗
- 引战、侮辱/贬损性评论，以及人身或政治攻击
- 公开或私下骚扰
- 未经明确许可，发布他人的私人信息（如地址）
- 其他在专业环境中通常被认为不适当的行为

## 我们的责任

项目维护者有责任澄清可接受行为的标准，并对任何不可接受的行为采取适当且公平的纠正措施。

项目维护者有权利和责任删除、编辑或拒绝与行为准则不一致的评论、提交、代码、Wiki 编辑、Issue 和其他贡献，并在适当时沟通其审核决定的原因。

## 适用范围

本行为准则适用于所有项目空间，也适用于个人在公共场合代表项目或社区时的情况。

## 执行

可以通过在 GitHub Issue 中 @项目维护者来报告辱骂、骚扰或其他不可接受的行为。所有投诉都将被审查和调查，并被视为必要和适当的回应。项目团队有义务对事件报告者保密。
